About:

The Applicates fecthes flag images along with some country details from Rest Countries API and fetches the summaries of the countries from MediaWiki API. The Application is built completely on Angular Framework and fetches the data using the HTTPClient Module.

Tools Used:

Rest Countires: https://restcountries.eu/
WikiMedia API: https://en.wikipedia.org/api/rest_v1/

Angular Material: https://material.angular.io/"

Source Code : https://github.com/arjhn/angular1
